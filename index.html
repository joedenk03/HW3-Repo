Projects

ChessConnect

The ChessConnect is an automatic chessboard that can move pieces from within the board. It also interfaces with the popular online chess site Lichess.org to allow for remote play using a physical board and pieces.

OOO CPU Simulator

The goal of this project was to make a cycle accurate CPU pipeline simulator that, given a set of instructions, produced accurate register results and a cycle-tracking log. The simulator had out-of-order execution capabilites, branch prediction, and modularity for the various hardware components.
